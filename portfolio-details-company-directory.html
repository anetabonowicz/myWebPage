Company Directory

Company Directory is a mobile first development application but it works equally well on desktop PCs. It provides functionality to interact with a back end server database from a mobile or desktop based client that allows for creation, retrieval and modification of company personnel (CRUD).

Application is making use of following technologies:

● HTML / CSS (with Bootstrap framework)
● JavaScript
● JQuery DOM manipulation / AJAX
● PHP with JSON encoding / MySQL
● XAMPP
